# Digital Farm Management Portal

## Project Title
Development of a Digital Farm Management Portal for implementing Biosecurity measures in Pig and Poultry Farms

## Description
This project helps farmers manage pig and poultry farms using a digital portal. 
It tracks animal health, vaccination records, visitor logs, and farm sanitation to prevent disease spread.

## Features
- Animal health monitoring
- Vaccination tracking
- Visitor management
- Farm sanitation tracking
- Disease alerts

## Technologies
- HTML
- CSS
- JavaScript
- Python / PHP
- MySQL

## Purpose
To improve farm biosecurity and prevent disease outbreaks using a digital management system.
